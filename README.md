# Petfolio

Petfolio is a simple pet management API built with **.NET 8**.

## About

This project was created to practice a layered architecture in .NET.
It is organized into different projects for API, application rules, and communication models.

## What I built so far

- Pet registration
- Get all pets
- Get pet by id
- Update pet
- Delete pet
- Request and response models
- Swagger setup

## Tech stack

- .NET 8
- ASP.NET Core Web API
- Swagger

## Current status

The project is still in progress and currently uses simple logic without a database.

## Next step

My next step is to learn more about **databases** and then implement persistence in the project.

## Final note

This repository is part of my learning journey in .NET and layered architecture.
